# 🍽️ Kitchen Level P&L Dashboard

An interactive dashboard built using **Python, Pandas, Streamlit, and Plotly** to analyze kitchen/store-level operational and profitability metrics.

The dashboard enables users to monitor revenue performance, EBITDA trends, profitability distribution, and store-level operational insights through interactive filters and visualizations.

---

## 📌 Problem Statement

Businesses often deal with large volumes of operational and financial data spread across multiple entities such as stores or kitchens. Extracting actionable insights from raw data becomes difficult without a structured analytical view.

The objective of this dashboard is to transform raw operational data into meaningful KPIs and interactive insights for better decision-making.

---

## 🚀 Features

### Dynamic Filtering

* Store selection
* Zone filtering
* Month filtering
* Revenue Cohort filtering
* EBITDA Category filtering
* Status filtering

### KPI Tracking

* Total Net Revenue
* Kitchen EBITDA
* EBITDA %
* Total Orders
* Average Order Value (AOV)
* Gross Margin %

### Dashboard Insights

* Kitchen performance snapshot
* Top 10 stores by EBITDA
* Revenue cohort distribution
* Monthly revenue trend analysis

## ⚙️ Installation & Run Locally

Clone repository:

```bash
git clone https://github.com/yourusername/Kitchen-Level-PnL-Dashboard-Streamlit.git
```

Move into project directory:

```bash
cd Kitchen-Level-PnL-Dashboard-Streamlit
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run application:

```bash
streamlit run app.py
```
